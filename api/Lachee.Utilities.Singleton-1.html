    Awake()

    Called when the object is first initialized in the scene.

    protected virtual void Awake()

    OnApplicationQuit()

    When Unity quits, it destroys objects in a random order. In principle, a Singleton is only destroyed when application quits.

    If any script calls instance after it have been destroyed, it will create a buggy ghost object that will stay on the Editor scene when the player stops. This override prevents this from happening and must be called.
